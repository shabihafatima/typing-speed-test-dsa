# Typing Speed Test Game (DSA Project)

## Overview

This project is a console-based Typing Speed Test Game developed in C++. It demonstrates the practical implementation of Data Structures and Algorithms (DSA), Object-Oriented Programming (OOP), and File Handling.

## Technologies

- C++
- Data Structures & Algorithms (DSA)
- Object-Oriented Programming (OOP)
- File Handling

## DSA Concepts Used

- Linked List (User Management)
- Queue (Word Management)
- Stack (Mistake Tracking)

## Features

- User Registration & Login
- Forgot Password
- Multiple Difficulty Levels
- 60-Second Typing Test
- Words Per Minute (WPM) Calculation
- Accuracy & Streak Tracking
- Performance History
- Leaderboard
- File-Based Data Storage

## Learning Outcomes

- Practical implementation of Linked Lists, Queues, and Stacks
- File handling for persistent data storage
- Object-oriented design using classes and encapsulation
- Performance tracking using DSA concepts
